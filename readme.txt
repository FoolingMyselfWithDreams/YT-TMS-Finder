To use this tool, run "# TMS Finder.bat" or create a shortcut to it on your desktop.
Run "# Collect Song Fingerprints.bat" to add songs you want to find to the database,
or use "# Reset Fingerprint Database.bat" to delete all songs from the database.
"# Get Database Info" gives you more insight about the state of the database.

Refer to "README.md" for more info about this tool.
